Web-Ohjelmistokehitys

Web-ohjelmistokehityksestä minulla on kokemusta laajalti niin front-end, kuin back-end kehityksestä. Paikkatietotekniikatkaan eivät ole minulle vieraita.

Alemmaksi tälle sivulle olen koonnut referensseihin joitain vanhempia julkisia työnäytteitäni. Tuttuja tekniikoita ovat mm.

  • HTML5
  • CSS3, Bootstrap, (resonsiiviset käyttöliittymät)
  • Javascript, JQuery, AngularJS, Node.js, (yksikkötestaus Jasmine, Karma)
  • Java, Hibernate
  • PHP
  • MySQL, MS SQL Server
  • JSON
  • XML
  • REST
  • Webservicet/integraatiot
  • OpenLayers
  • Google Maps
  • jne...

.NET & C#

Itselleni suosikki ohjelmointikieli C# josta työnäytteinä esimerkiksi opinnäytetyöni (MVVM-mallin mukaisen WPF-sovelluksen yksikkötestaus) sekä Unity peliprojektit. Windows-työpöytäsovellusten ja ASP.NET web-sovellusten kehitys, ovat molemmat minulle tuttua.

Kokemusta em.tekniikoista on kertynyt työharjoittelun ja vakituisen työsuhteen myötä Savcor Oy:ssä (nykyisin Trimble). Tuttuja tekniikoita ovat mm.

  • C#
  • WPF, XAML
  • WinForms
  • DevExpress
  • Entity Framework
  • MySQL, MS SQL Server, Oracle SQL
  • MS Unit tests
  • ASP.NET Razor
  • Moq - Mocking library
  • MVVM- ja MVC-arkkitehtuurimallit
  • jne...

3D-Mallinnus ja media

Vaikkakin ohjelmistokehitys on minulle mieleisintä hommaa, tykkään myös vastapainona 3D-mallinnuksesta sekä median käsittelystä (kuvakäsittely, audion ja videon editointi).

3D-mallinnukseen käyttämäni ohjelmat ovat 3ds Max ja SketchUp sekä median käsittelyyn tuttuja ohjelmia ovat Adoben tuotteet (Photoshop, Illustrator, Auditio, After effects, Premiere)
